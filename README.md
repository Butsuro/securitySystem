Steps to run programme:

1) After downloading, extract the folder and open on your IDE.
2) Open your Powershell terminal and change directory to frontend folder
3) Install node modules (use: npm install)
4) Install Tailwind CSS
   - Run: npm install -D tailwindcss postcss autoprefixer
   - Run: npx tailwindcss init -p
5) Start server (use: npm start)
